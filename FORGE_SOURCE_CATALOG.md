# FORGE_SOURCE_CATALOG — S&C Knowledge Source Registry

> Every source mined for FORGE patterns. No opinions — only metadata.
> Sources classified by credibility tier. Real programs weighted above theory.

---

## Classification Rules

- **Credibility Score** (0.0–1.0): based on sample size, sport specificity, peer review, practical application.
- **Priority Level** (1–5): P1 = must-extract, P5 = reference-only.
- **Type**: Book / Certification / Real Program / Research / Manual.
- **Organization**: The program, institution, or governing body behind the source.
- **Sport**: The primary sport context. "General" = no single sport focus.

## Source-to-Pattern Traceability

Every pattern in FORGE_PATTERN_DATABASE.md must reference one or more sources from this catalog.

```
Format in pattern entries:
  source: [Source Title] → [FORGE_SOURCE_CATALOG.md reference]
```

Sources marked as "Unnamed (FORGE corpus)" are programs reverse-engineered into the FORGE system during initial development. Their specific team/club identifiers are removed by design to prevent methodology-branding bias.

## Source Quality Metrics Used

| Metric | Definition | Weight |
|--------|-----------|--------|
| Sample Size | Number of athletes/programs the source represents | 30% |
| Peer Review | Reviewed by ≥2 domain experts or published in peer-reviewed journal | 25% |
| Sport Specificity | Directly applicable to ≥1 sport in FORGE | 20% |
| Practical Application | Used in real coaching environment, not just theoretical | 15% |
| Recency | Published/used within last 10 years | 10% |
