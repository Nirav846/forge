# FORGE Substitution Matrix V1

> Substitution ladders for every exercise family. Use when an athlete lacks equipment, has an injury, or needs a progression / regression within the same movement family.

---

## Rules

1. Substitutions stay **inside their family** — swapping Knee Dominant for Hip Dominant is program design, not substitution.
2. Progressions move **simple → complex** (load, coordination, stability).
3. Equipment limitations: if an athlete has no barbell, jump to the barbell-free rung.
4. Injury-friendly alternatives marked with **(inj)** — these offload the affected tissue while preserving the pattern.
5. Ladder positions (L1-L5) are relative within each family. Not all families have 5 rungs — some have fewer, some more.

## Knee Dominant Ladder

| Level | Double Leg | Single Leg | Notes |
|-------|-----------|------------|-------|
| L1 | Bodyweight Squat **(inj)** | Split Squat (assisted) | Minimal load, patterning only |
| L2 | Goblet Squat | Step-Up | Load added, still controlled |
| L3 | Barbell Back Squat | Bulgarian Split Squat | Core strength level |
| L4 | Barbell Front Squat | Barbell Reverse Lunge | Quad-dominant emphasis |
| L5 | Paused / Tempo Front Squat | Pistol Squat | Maximal demand |

**Equipment fallback:** No barbell? Goblet Squat (L2) + Bulgarian Split Squat (L3) covers most needs.

**Injury notes:**
- Low back pain → Goblet Squat, Box Squat, Leg Press, Step-Ups
- Knee pain (patellar) → Shallow range Step-Ups, Leg Press (reduced ROM), avoid deep squats
- ACL return → Step-Ups → Bulgarian Split Squat (shallow) → Full range over 12+ weeks

## Hip Dominant Ladder

| Level | Double Leg | Single Leg | Notes |
|-------|-----------|------------|-------|
| L1 | Glute Bridge **(inj)** | Single-Leg Glute Bridge | Low load, control focus |
| L2 | Kettlebell Deadlift | Single-Leg RDL (floor touch) | Hinge patterning |
| L3 | RDL | Weighted Single-Leg RDL | Hamstring loading |
| L4 | Barbell Good Morning | 45° Back Extension (single leg) | Erector + hamstring |
| L5 | Conventional Deadlift | Nordic Hamstring Curl | Max posterior chain |

**Equipment fallback:** No barbell? Kettlebell Deadlift (L2) + Single-Leg RDL (L3). Trap Bar Deadlift replaces Conventional if back-sensitive.

**Injury notes:**
- Low back / disc → Trap Bar Deadlift, Kettlebell Deadlift, Hip Thrust, avoid Conventional Deadlift
- Hamstring strain (acute) → Isometric Glute Bridge → Single-Leg RDL (pain-free ROM only) → slow eccentrics
- Hamstring strain (return to run) → Nordic Hamstring Curl (eccentric emphasis)

## Push Ladder (Horizontal + Vertical Combined)

| Level | Horizontal | Vertical | Notes |
|-------|-----------|----------|-------|
| L1 | Wall Push-Up **(inj)** | Landmine Press (half-kneeling) | Minimal load, scapular control |
| L2 | Push-Up | Landmine Press (standing) | Bodyweight control |
| L3 | Dumbbell Bench Press | Dumbbell Overhead Press | Unilateral, shoulder-friendly |
| L4 | Barbell Bench Press | Barbell Overhead Press | Max load horizontal / vertical |
| L5 | Incline Barbell Bench Press | Push Press / Power Jerk | Incline variation / hip drive added |

**Equipment fallback:** No bench? Floor Press + Landmine Press covers all. No rack? Dumbbells only (L3 covers both planes).

**Injury notes:**
- Shoulder impingement → Landmine Press, Dumbbell Floor Press, Neutral-Grip DB Press, avoid barbell
- AC joint → Dumbbell only, half-ROM, no barbell bench
- Elbow tendinopathy → Neutral-grip variations, reduce volume, avoid close-grip bench
- Return from shoulder injury → Wall Push-Ups → Incline Push-Ups → Push-Ups → Landmine Press

## Pull Ladder (Horizontal + Vertical Combined)

| Level | Horizontal | Vertical | Notes |
|-------|-----------|----------|-------|
| L1 | Band Row **(inj)** | Band Lat Pulldown | Low threshold, activation |
| L2 | Inverted Row | Lat Pulldown | Bodyweight / controlled |
| L3 | Dumbbell Row | Pull-Up / Chin-Up | Unilateral / bodyweight max |
| L4 | Barbell Row | Weighted Pull-Up | Max load each plane |
| L5 | Pendlay Row (explosive) | Muscle-Up | Explosive + combo |

**Equipment fallback:** No pull-up bar? Lat Pulldown cable / band version. No barbell? Dumbbell Row + Inverted Row. No cable? Bands cover all vertical and most horizontal.

**Injury notes:**
- Shoulder labrum → Neutral-grip pulls, avoid wide grip, avoid heavy weighted pull-ups
- Elbow tendinopathy (medial) → False grip, neutral grip, reduce chin-up volume, band-assisted
- Low back (during rows) → Chest-Supported Row, Seal Row, Inverted Row

## Plyometric Ladder

| Level | Bilateral | Unilateral | Directional |
|-------|-----------|------------|-------------|
| L1 | Pogo Jump **(inj)** | Single-Leg Pogo (low) | Lateral Pogo (low) |
| L2 | Squat Jump | Broad Jump (takeoff focus) | Broad Jump |
| L3 | Countermovement Jump | Single-Leg Broad Jump | Lateral Bound |
| L4 | Box Jump | Bounding | Hurdle Hop |
| L5 | Depth Jump | Single-Leg Depth Jump | Depth + Sprint Combination |

**Equipment fallback:** No box? Use line jumps, broad jumps, pogo variations. No hurdles? Cone hops.

**Injury notes:**
- Patellar tendinopathy → Avoid depth jumps, use pogo + squat jumps, isometric squat holds between reps
- Achilles tendinopathy → Avoid heavy plyos, pogo (low amplitude), seated calf raises in warm-up
- ACL return (late stage) → Low-amplitude pogos → SL bounds (forward) → lateral bounds (week 12+) → depth jumps (week 16+)

## Ballistic Ladder

| Level | Pull Variant | Push Variant | Notes |
|-------|-------------|--------------|-------|
| L1 | Jump Shrug **(inj)** | Med Ball Chest Pass | Patterning, low load |
| L2 | High Pull (hang) | Med Ball Overhead Throw | Explosive pull / push |
| L3 | Hang Clean | Med Ball Slam | Catch mechanics / full body |
| L4 | Power Clean | Rotational Med Ball Slam | Floor start / rotation added |
| L5 | Clean + Jerk | Overhead Med Ball Slam (max effort) | Full competition lift / max intent |

**Equipment fallback:** No barbell / bumpers? Med Ball covers all levels (L1-L5 with ball). No med ball? Band explosive pulls (band high pull).

**Injury notes:**
- Wrist / hand → Med Ball only, no barbell catch
- Low back → Jump Shrug, High Pull (no catch), avoid full clean until spasm-free
- Shoulder (overhead athlete) → Avoid behind-neck variants, use snatch-grip pulls instead of full snatch

## Rotational Ladder

| Level | Loaded Rotation | Anti-Rotation | Dynamic |
|-------|----------------|---------------|---------|
| L1 | Band Rotational Chop **(inj)** | Band Anti-Rotation Press | Half-Kneeling Landmine |
| L2 | Cable Rotational Row | Pallof Press (hold) | Standing Landmine Rotation |
| L3 | Med Ball Rotational Throw | Single-Leg Pallof Press | Cable Chop (high → low) |
| L4 | Landmine Rotation (heavy) | Walking Pallof Press | Med Ball Overhead Rotational Slam |
| L5 | Full Rotational Med Ball Slam | Single-Leg + Band Resisted Pallof | Rotational Slam + COD |

**Equipment fallback:** No cable? Bands cover L1-L3. No med ball? Landmine alone covers L1-L4.

**Injury notes:**
- Lumbar spine (acute) → Anti-rotation only (Pallof), no loaded rotation
- Hip / groin → Half-kneeling variations, band anti-rotation, no standing rotation
- Return from oblique strain → Anti-rotation (Pallof) → gentle band chops → full rotation over 4-6 weeks

## Core Ladder

| Level | Anti-Extension | Anti-Rotation | Flexion / Dynamic |
|-------|---------------|---------------|-------------------|
| L1 | Dead Bug **(inj)** | Pallof Press (hold) | Posterior Pelvic Tilt |
| L2 | Plank | Single-Leg Pallof | Reverse Crunch |
| L3 | RKC Plank | Cable Anti-Rotation Press | Hanging Knee Raise |
| L4 | Ab Wheel Rollout (kneeling) | Walking Pallof | Hanging Leg Raise |
| L5 | Ab Wheel Rollout (standing) | Single-Leg + Overhead Pallof | Toes to Bar |

**Equipment fallback:** No ab wheel? Stir the Pot (stability ball). No cable / bands? All L1-L5 can be done with bodyweight except Walking Pallof.

**Injury notes:**
- Low back (any) → Dead Bug, Pallof, avoid situps, avoid rollouts until pain-free
- Neck / cervical → Keep hands at sides, avoid hands-behind-head, no situps
- Hip flexor dominant → Dead Bug (keep lumbar flat), avoid hanging leg raises if anterior pelvic tilt

## Sprint / COD Ladder

| Level | Acceleration | Max Velocity | Change of Direction |
|-------|-------------|-------------|---------------------|
| L1 | Wall Lean Fall **(inj)** | A-Skip | March to Stop |
| L2 | 10m Sled Push / Pull | A-Run | Deceleration to Stop |
| L3 | 10m Block Start | Flying 10m | Pro Shuttle (5-10-5) |
| L4 | 20m Acceleration (timed) | Flying 20m | 3-Cone Drill (L-Drill) |
| L5 | Resisted Sprint (heavy sled) | Flying 30m (max velocity) | T-Drill (reactive) |

**Equipment fallback:** No sled? Hill sprints, heavy band resisted. No timing gates? Coded effort. No cones? Landmarks.

**Injury notes:**
- Hamstring (return) → Wall Lean → A-Skip → 50% effort 10m → 80% → full. No max velocity until 12+ weeks pain-free
- Achilles / calf → Sled pushes (concentric emphasis), avoid max V, avoid bounding
- Groin / adductor → Limit COD volume, linear-only, lateral lunges for prep
- ACL return → Linear acceleration (week 8+) → deceleration (week 12+) → COD (week 16+) → reactive COD (week 20+)

## Carry Ladder

| Level | Bilateral | Unilateral | Advanced |
|-------|-----------|------------|----------|
| L1 | Farmer's Walk (light) **(inj)** | Suitcase Carry (light) | Bear Hug Carry |
| L2 | Farmer's Walk (moderate) | Suitcase Carry (moderate) | Front Rack Carry |
| L3 | Trap Bar Carry (heavy) | Waiter's Walk | Overhead Carry (bilateral) |
| L4 | Farmer's Walk (max load) | Single-Arm Overhead Carry | Zercher Carry |
| L5 | Yoke Walk | Suitcase Carry (max) | Front Rack + Overhead Combo |

**Equipment fallback:** No farmer's handles? Dumbbells cover L1-L4. No yoke? Barbell in front rack or behind back.

**Injury notes:**
- Low back → Farmer's Walk (brace hard), avoid Zercher, avoid Yoke
- Grip issues → Straps, mixed grip, or Bear Hug Carry
- Shoulder (unilateral) → Bilateral only, no Waiter's Walk, no Overhead Carry
- Core in rehab → Suitcase Carry (anti-lateral flexion stimulus is low-load, high-value)

## Cross-Family Substitution (Emergency Only)

When an athlete genuinely cannot train a family, use these as last-resort swaps:

| Missing Family | Best Substitute | Why |
|----------------|----------------|-----|
| Knee Dominant (bilateral) | Single Leg Knee Dominant | Same dominant pattern, single leg |
| Knee Dominant (any) | Hip Dominant | Different pattern but largest lower-body overlap |
| Hip Dominant (bilateral) | Single Leg Hip Dominant | Same hinge pattern |
| Hip Dominant (any) | Knee Dominant (RDL-stance squat) | Weak swap — only if no hinge possible |
| Horizontal Push | Vertical Push (incline reduces difference) | Different plane but pressing pattern |
| Horizontal Pull | Vertical Pull (neutral grip) | Different plane but pulling pattern |
| Vertical Push | Horizontal Push (incline bench) | Different plane, similar force angle |
| Vertical Pull | Horizontal Pull (chest-supported row, high elbow) | Different plane, similar lat engagement |
| Plyometric | Ballistic (lighter load, faster intent) | Both are explosive — ballistic has load |
| Ballistic | Plyometric (loaded jump) | Explosive output — plyometric is bodyweight |
| Sprint / COD | Plyometric (bounds, lateral bounds) | Movement quality + power transfer |
| Rotational | Anti-rotation Core | Builds the stiffness rotation needs |
| Carry | Core (loaded carries are core + grip) | Overlapping demand — grip and bracing |
| Core | Carry (any heavy carry) | Anti-extension + bracing overlap |
| Accessory / Prehab | Nothing — do not substitute | Do the prehab or accept the gap |

> **ponytail: cross-family substitutions are lossy.** The matrix tells you the least-bad option, not a good one. Preferred path: regress inside the family, not swap out of it.

## Quick Reference: Equipment-Limited Athlete

| Equipment Available | Best Families to Programme |
|--------------------|---------------------------|
| Bodyweight only | Core, Plyometric, Sprint/COD, Single Leg Knee (Split Squat) |
| Resistance Bands | Core (Pallof), Rotational (Chops), Pull (Band Row), Prehab |
| Dumbbells (only) | All families except Ballistic (use med ball / jump) and Sprint |
| Dumbbells + Bench | All families except Olympic lifting variants |
| Barbell + Rack | All families |
| Barbell + Rack + Bumpers | All families + Olympic lifting |
| Kettlebells | All families — KB swing replaces ballistic, KB squat replaces barbell squat |
| No squat rack | Front squat / Zercher squat from floor, box squat from chair |
| Travel / hotel | Band + bodyweight covers Core, Pull, Push (low load), Sprint mechanics |

## Using This Matrix

1. **Athlete has an injury?** Look up the family → check injury notes → pick the highest injury-friendly rung.
2. **Athlete lacks equipment?** Look up the family → check equipment fallback → pick the highest rung the equipment allows.
3. **Athlete needs progression?** Move up one rung on the ladder. Stay 2+ weeks at each rung.
4. **Athlete cannot do a family at all?** Use cross-family table — but note the loss in the programme notes.
5. **Unknown athlete (database seed)?** Seed at L2-L3 for each family — the most commonly prescribed range.
