## Key Session Structure Details from New Sources

### Jack Tyler Rugby In-Season Templates:
- Plan 1 (Lower-Upper-Full, 2-day): Day A = Mobility → Power → Bilateral + Unilateral Lower → Core, Day B = Mobility → Horizontal Push → Horizontal Pull → Vertical Push → Vertical Pull → Core
- Plan 2 (Full-body Strength and Power, 2-day): Day A & B both: Power → Bilateral Lower → Unilateral Lower → Horizontal Push → Horizontal Pull → Core
- Plan 3 (Full-body Strength Focus, 2-day): Day A & B both: Bilateral Lower → Horizontal Push → Horizontal Pull → Core (no power slot)
- Plan 4 (Lower-Upper-Full V2, 2-day): Day A = Power → Bilateral Lower → Horizontal Push → Core, Day B = Power → Unilateral Lower → Horizontal Pull → Vertical Pull

### StrengthLog Sprint Program Structure (12-week):
- Phase 1 (Weeks 1-4): Accumulation - higher volume, moderate intensity
- Phase 2 (Weeks 5-8): Intensification - lower volume, higher intensity
- Phase 3 (Weeks 9-12): Pre-Competition - sport-specific, peaking
- Gym sessions: 3x/week full body
- Speed sessions: 2-3x/week track work

### StrengthLog Badminton Program (12-week):
- Phase 1 (Weeks 1-4): Strength Endurance - 3x12-15, 60-90s rest
- Phase 2 (Weeks 5-8): Pure Strength - 3x6-8, 2-3min rest
- Phase 3 (Weeks 9-12): Speed & Power - heavy + explosive mixed
- 3x/week full body, ~60min sessions
- Session structure: Warm-up → Power → Bilateral Lower → Horizontal Push → Horizontal Pull → Core

### Badminton Insight 4 Phases:
- Strength Endurance (4-6wks): higher reps (12-15), shorter rest, compound focus
- Pure Strength (4-6wks): lower reps (4-6), longer rest, heavier loads
- Speed & Power (4-6wks): contrast training, complex pairs, explosive focus
- Power Endurance (4-6wks): circuit-based, sport-specific conditioning

### IMG Academy Tennis S&C (2-3x/week):
- Session: Dynamic Warm-up → Agility → Power → Strength → Core → Conditioning
- Year-round programming with periodized phases

### Kovacs Institute Adult Tennis L1 (6-week):
- Session: Dynamic Warm-up → Linear Speed → Lateral Agility → Power → Strength → Elasticity → Core
- 2-3x/week depending on competitive schedule

### Damon Bednarski Cricket Pre-Season (12-week, 3-phase):
- Phase 1 (Weeks 1-4): Strength Foundation - bilateral emphasis, moderate reps
- Phase 2 (Weeks 5-8): Strength & Power - mixed methods, heavier loads
- Phase 3 (Weeks 9-12): Power & Speed - lighter loads, faster movements, plyometrics

### Nathan Kiely Forging Fast Bowlers (16-week):
- Focus: strength durability for fast bowling workload
- Emphasizes: eccentric control, isometric work, unilateral stability
- Key exercises: single-leg RDL, Copenhagen plank, heavy bilateral lifts

### BokSmart Programs (SA Rugby):
- Senior Adult: General Warm-up → Injury Prevention → Power → Strength → Conditioning → Cool Down
- U16 Youth: General Warm-up → Movement Skills → Strength → Conditioning → Cool Down
- Evidence-based, age-appropriate progression
