# FORGE Blueprint Catalog V1

> 14 universal training blueprints distilled from 58-program research corpus.
> These are reusable session structures — not sport-specific programs. Sport arrives as constraint, not identity.

---

## How To Read

| Field | Meaning |
|-------|---------|
| **Purpose** | What this blueprint accomplishes |
| **Typical Athlete** | Who uses this most |
| **Frequency** | Sessions per week |
| **Session Structure** | How the session layering works |
| **Slot Order** | Priority sequence for exercise families |
| **Mandatory Families** | Must be present every session |
| **Optional Families** | Rotated in as needed |
| **Common Progressions** | How this blueprint evolves over a block |
| **When NOT to use** | Contraindications |

---

## 1. Full Body Strength

| Field | Value |
|-------|-------|
| **Purpose** | General strength accrual across all major movement patterns. Highest frequency of each pattern per week. |
| **Typical Athlete** | Beginner-intermediate, off-season general population, youth athletes (14-17), sport-generalists |
| **Frequency** | 3-4 sessions / week |
| **Session Structure** | 2-3 strength families + accessory finisher. Full body every session. |
| **Slot Order** | 1. Ballistic / explosive (if included) → 2. Knee Dominant → 3. Horizontal Push → 4. Hip Dominant / Pull → 5. Core |
| **Mandatory Families** | 1 Knee Dominant, 1 Hip Dominant, 1 Horizontal Push, 1 Horizontal Pull |
| **Optional Families** | Core, Accessory, Vertical Push, Vertical Pull |
| **Common Progressions** | Linear progression → DUP (daily undulating periodisation) → Block periodisation |
| **When NOT to use** | Advanced athletes needing specific peak, late pre-season, in-season maintenance phase |

---

## 2. Strength + Power

| Field | Value |
|-------|-------|
| **Purpose** | Hybrid strength-power development. Heavy strength + explosive work in same session. |
| **Typical Athlete** | Intermediate-advanced, field sport off-season, track & field, rugby pre-season |
| **Frequency** | 4-5 sessions / week |
| **Session Structure** | Explosive primer → heavy compound → lighter explosive variant → accessory |
| **Slot Order** | 1. Ballistic → 2. Knee Dominant (heavy) → 3. Push / Pull → 4. Hip Dominant (explosive variant) → 5. Core |
| **Mandatory Families** | 1 Ballistic, 1 Knee Dominant, 1 Hip Dominant, 1 Push |
| **Optional Families** | Carry, Sprint/COD, Rotational, Core |
| **Common Progressions** | Hinge-first waves: Strength block → Strength-Power block → Power block |
| **When NOT to use** | Athletes with poor movement quality, early post-injury, hypertrophy-deficient athletes needing mass first |

---

## 3. Strength + Conditioning

| Field | Value |
|-------|-------|
| **Purpose** | Strength maintenance + energy system development. Common in pre-season / GPP blocks. |
| **Typical Athlete** | Team sport, pre-season, returning from off-season break |
| **Frequency** | 3-4 sessions / week |
| **Session Structure** | Strength circuit or paired sets → conditioning finisher (strongman, runs, interval) |
| **Slot Order** | 1. Compound strength → 2. Superset accessory → 3. Conditioning finisher → 4. Core |
| **Mandatory Families** | 1 Knee Dominant or 1 Hip Dominant, 1 Push, 1 Pull |
| **Optional Families** | Carry, Core, Accessory, Sprint/COD |
| **Common Progressions** | Decrease rest → increase conditioning volume → separate strength / conditioning to different days |
| **When NOT to use** | Athletes needing peak strength (conditioning interferes), low recovery capacity |

---

## 4. Power + Speed

| Field | Value |
|-------|-------|
| **Purpose** | Maximise rate of force development + velocity. Low volume, high intensity, full recovery. |
| **Typical Athlete** | Advanced field sport, track sprinters, court sport in-season, peaking phase |
| **Frequency** | 3-4 sessions / week |
| **Session Structure** | Speed primer → plyometric / ballistic → light strength (velocity focus) |
| **Slot Order** | 1. Sprint / COD → 2. Plyometric → 3. Ballistic → 4. Hip Dominant (explosive) → 5. Core |
| **Mandatory Families** | 1 Sprint/COD, 1 Plyometric, 1 Ballistic |
| **Optional Families** | Hip Dominant, Horizontal Push, Core |
| **Common Progressions** | Increase intensity (lower reps, higher load / speed) → add reactive elements → reduce volume |
| **When NOT to use** | Strength-deficient athletes, early post-injury, hypertrophy phase |

---

## 5. Upper / Lower Split

| Field | Value |
|-------|-------|
| **Purpose** | Higher per-session volume per region. Two lower sessions, two upper sessions per microcycle. |
| **Typical Athlete** | Intermediate-advanced bodybuilding, strength athletes, anyone tolerating higher per-session volume well |
| **Frequency** | 4 sessions / week (Upper → Lower → Upper → Lower) |
| **Session Structure** | Compound → accessory → isolation. Lower: squat → hinge. Upper: horizontal → vertical. |
| **Slot Order** | **Lower**: 1. Plyometric → 2. Knee Dominant → 3. Hip Dominant → 4. Carry → 5. Core. **Upper**: 1. Vertical Push → 2. Horizontal Push → 3. Vertical Pull → 4. Horizontal Pull → 5. Accessory |
| **Mandatory Families** | **Lower**: 1 Knee Dominant, 1 Hip Dominant. **Upper**: 1 Horizontal Push, 1 Horizontal Pull |
| **Optional Families** | Carry, Core, Accessory, Rotational, Vertical Push/Pull |
| **Common Progressions** | Linear → conjugate / concurrent periodisation |
| **When NOT to use** | Athletes needing full body frequency (youth, low training age), very long season with limited sessions |

---

## 6. Power Maintenance

| Field | Value |
|-------|-------|
| **Purpose** | Retain power qualities with minimal fatigue during season. Low dose, high stimulus. |
| **Typical Athlete** | In-season team sport, competing court sport, mid-competition phase |
| **Frequency** | 1-2 sessions / week |
| **Session Structure** | Activation → one explosive movement → one plyometric → accessory |
| **Slot Order** | 1. Sprint / COD (short) → 2. Ballistic (1-2 lifts) → 3. Plyometric (low volume) → 4. Core / Accessory |
| **Mandatory Families** | 1 Ballistic, 1 Plyometric |
| **Optional Families** | Sprint/COD, Hip Dominant, Core, Accessory |
| **Common Progressions** | Increase weight on ballistic → reduce volume further → move to pre-match protocol only |
| **When NOT to use** | Off-season, de-loaded recovery weeks, early return from injury |

## 7. Youth Foundation (U14-U17)

| Field | Value |
|-------|-------|
| **Purpose** | Build movement literacy, general strength, and athleticism. No specialisation. Fun + competence. |
| **Typical Athlete** | Adolescents (13-17), multi-sport, beginner training age |
| **Frequency** | 2-3 sessions / week |
| **Session Structure** | Warm-up (coordination) → teach one pattern → circuit → game / competition |
| **Slot Order** | 1. Sprint / COD (low intensity) → 2. Teach pattern (squat / hinge / push) → 3. Superset circuit → 4. Core → 5. Fun finisher |
| **Mandatory Families** | 1 Knee Dominant or Hip Dominant, 1 Push, 1 Pull |
| **Optional Families** | All — rotate freely. Prioritise variety over specialisation. |
| **Common Progressions** | Increase pattern complexity → add load → reduce circuit, increase compound focus |
| **When NOT to use** | Athletes 18+ (move to adult blueprint), sport-specialised juniors in off-season |

---

## 8. Court Sport Athletic Development

| Field | Value |
|-------|-------|
| **Purpose** | Multi-directional power, deceleration, change of direction, and lateral strength. |
| **Typical Athlete** | Tennis, badminton, basketball, volleyball, netball — competition / pre-season |
| **Frequency** | 3-4 sessions / week |
| **Session Structure** | COD primer → plyometric → strength (single-leg emphasis) → rotational → accessory |
| **Slot Order** | 1. Sprint / COD → 2. Plyometric (lateral + vertical) → 3. Single Leg Knee Dominant → 4. Rotational → 5. Pull → 6. Core / Prehab |
| **Mandatory Families** | 1 Sprint/COD, 1 Single Leg Knee Dominant, 1 Push, 1 Rotational |
| **Optional Families** | Hip Dominant, Ballistic, Carry, Core, Accessory |
| **Common Progressions** | Increase COD speed → single-leg strength → reactive rotational power |
| **When NOT to use** | Linear-only sport (track, cycling), early rehab phase |

---

## 9. Rugby Off-Season

| Field | Value |
|-------|-------|
| **Purpose** | Build collision-ready strength, power, and mass. Contact preparation with conditioning. |
| **Typical Athlete** | Rugby union / league, American football, contact sport — off-season phase |
| **Frequency** | 4-5 sessions / week |
| **Session Structure** | Power primer → heavy compound → strongman / carry → contact prep → conditioning |
| **Slot Order** | 1. Ballistic → 2. Knee Dominant (heavy) → 3. Horizontal Pull / Push → 4. Carry → 5. Sprint / COD (sled) → 6. Neck / Prehab |
| **Mandatory Families** | 1 Ballistic, 1 Knee Dominant, 1 Horizontal Push, 1 Horizontal Pull, 1 Carry |
| **Optional Families** | Sprint/COD, Core, Rotational, Accessory |
| **Common Progressions** | Increase strength intensity → increase carry load → add contact-specific rotational → taper into pre-season |
| **When NOT to use** | In-season, non-contact sport, early off-season recovery phase (first 2-4 weeks off) |

## 10. Sprint Development

| Field | Value |
|-------|-------|
| **Purpose** | Max velocity and / or acceleration improvement. Strength underpins but speed is priority. |
| **Typical Athlete** | Track sprinters (100-400m), field sport speed block, returning from speed loss |
| **Frequency** | 3-4 sessions / week (speed + strength combo) |
| **Session Structure** | Sprint session (accel / max V) → plyometric → strength (hip dominant) → accessory |
| **Slot Order** | 1. Sprint / COD (main session) → 2. Plyometric → 3. Hip Dominant → 4. Knee Dominant (lighter) → 5. Core / Accessory |
| **Mandatory Families** | Sprint/COD (2-3 variations), 1 Hip Dominant, 1 Plyometric |
| **Optional Families** | Knee Dominant, Horizontal Pull, Core, Accessory |
| **Common Progressions** | Increase sprint volume / intensity → add resisted sprint → complex training (heavy squat + sprint) |
| **When NOT to use** | Injury return (Achilles, hamstring), technique-deficient, pre-meet taper |

## 11. Hypertrophy / Mass Accrual

| Field | Value |
|-------|-------|
| **Purpose** | Maximise muscle cross-sectional area. Higher volume, moderate loads, shorter rest. |
| **Typical Athlete** | Bodybuilding phase, underweight athlete, post-rehab mass return, off-season |
| **Frequency** | 4-6 sessions / week (PPL / Upper-Lower) |
| **Session Structure** | Compound → isolation → pump work. Higher sets (15-20+), lower rest (60-90s). |
| **Slot Order** | 1. Compound (lower rep) → 2. Accessory (higher rep) → 3. Isolation / Pump → 4. Core |
| **Mandatory Families** | 1 Knee Dominant, 1 Hip Dominant, 1 Push, 1 Pull |
| **Optional Families** | All — can rotate more freely. High volume across all families. |
| **Common Progressions** | Volume escalation (sets) → tempo changes → metabolic stress techniques (drops, rest-pause) |
| **When NOT to use** | In-season (unless severely underweight), pre-competition, speed / power peaking phases |

---

## 12. Return to Sport (Foundation)

| Field | Value |
|-------|-------|
| **Purpose** | Rebuild movement capacity post-injury. Low intensity, high control, no pain. |
| **Typical Athlete** | Post-rehab (cleared by PT), ACL return, hamstring return, general injury |
| **Frequency** | 3-4 sessions / week |
| **Session Structure** | Activation → movement patterning → low-load strength → proprioception → core |
| **Slot Order** | 1. Activation / Prehab → 2. Movement Pattern (regressed version) → 3. Single Leg (if applicable) → 4. Core → 5. Low-level COD (late phase) |
| **Mandatory Families** | 1 Accessory/Prehab, 1 regressed Knee or Hip Dominant, 1 Core |
| **Optional Families** | Single Leg, Sprint/COD (late stage), Rotational, Carry (light) |
| **Common Progressions** | Pain-free → load increase → complexity → sport-specific → full return |
| **When NOT to use** | Unreleased by medical, active pain, acute injury phase |

---

## 13. Deload / Active Recovery

| Field | Value |
|-------|-------|
| **Purpose** | Systemic recovery, movement practice, connective tissue health. Intent is not stimulus — it's maintenance. |
| **Typical Athlete** | Any athlete at end of block, post-competition, high fatigue, returning from illness |
| **Frequency** | 2-3 sessions / week |
| **Session Structure** | Mobility → light circuit → prehab → stretch |
| **Slot Order** | 1. Mobility / Activation → 2. Light Strength (50-60% 1RM, 2-3 RIR) → 3. Core → 4. Prehab |
| **Mandatory Families** | None minimum. Move well, don't strain. |
| **Optional Families** | Accessory, Core, Carry (light), Prehab |
| **Common Progressions** | N/A — this IS the regression. Exit to appropriate blueprint. |
| **When NOT to use** | When athlete needs stimulus (deloading too long risks detraining) |

## 14. Mixed Modal (General Physical Preparedness)

| Field | Value |
|-------|-------|
| **Purpose** | Broad athletic capacity. Strength, power, speed, endurance — nothing maximised, everything developed. |
| **Typical Athlete** | Off-season multi-sport, beginner athlete, military / tactical, general fitness |
| **Frequency** | 4-5 sessions / week |
| **Session Structure** | Varied daily. Full body strength → cardio / strongman / field work → varied accessory. |
| **Slot Order** | Rotates daily. No fixed slot order — cycles through families over the week. |
| **Mandatory Families** | All families hit at least 1-2x per week. Rotational priority lower. |
| **Optional Families** | None — all are mandatory at some frequency |
| **Common Progressions** | Increase specificity → narrow families → move toward sport-specific blueprint |
| **When NOT to use** | Sport-specialised athlete in-season, advanced needing peaking, any block requiring specificity |
