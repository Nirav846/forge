# FORGE Coaching Credibility Audit

> Multi-sport audit: elite S&C, rugby, cricket, tennis, youth development perspectives.
> Files reviewed: Exercise Ontology V1, Blueprint Catalog V1, Substitution Matrix V1, Pattern Library, Movement Frequency Report, Coaching Patterns Summary.

---

## 1. Exercise Ontology Audit

### DUPLICATES

| Exercise | Families | Problem |
|----------|----------|---------|
| Bulgarian Split Squat | 1 (DLKD) + 3 (SLKD) | Same exercise listed twice. Keep it in SLKD only. DLKD entry is redundant. |
| Single-Leg RDL | 2 (DL Hip) + 4 (SL Hip) | Appears in both families. Keep in SL Hip only. DL Hip entry clutters the table. |
| 45° Back Extension | 2 (DL Hip) + 4 (SL Hip) | Bilateral and unilateral versions listed in two families. Acceptable but the bilateral version should list single-leg as progression, not a separate entry. |
| Pallof Press | 12 (Rotational) + 14 (Core) | Listed in both. Pallof is anti-rotation core work — keep it in Core only, remove from Rotational. |

### INCORRECT FAMILY ASSIGNMENTS

| Exercise | Current Family | Should Be | Reason |
|----------|---------------|-----------|--------|
| Face Pull | 7. Vertical Push | 6. Horizontal Pull or 15. Accessory/Prehab | Face Pull is a horizontal scapular retraction + external rotation drill. It is NOT a vertical push. This is a basic anatomy error a coach spots immediately. |
| Bulgarian Split Squat | 1. Double Leg Knee Dominant | 3. Single Leg Knee Dominant | It's a single-leg exercise despite having two feet on the ground. The working leg is in single-leg stance. |

### INCORRECT PROGRESSIONS / REGRESSIONS

| Exercise | What's Wrong | Fix |
|----------|-------------|-----|
| Hack Squat → Barbell Back Squat | Hack Squat is more isolated than Back Squat. Progression is backwards. | Regression: Leg Press → Hack Squat → Barbell Back Squat (progression). Swap the chain. |
| Paused Back Squat → Back Squat (no pause) | Progression says "Barbell Back Squat (no pause)" — a paused squat is an intensification, not something you progress *to* from a regular squat. | Progression: Tempo Back Squat → Paused Back Squat. Regression: Barbell Back Squat. Currently reversed. |
| Tempo Back Squat → Paused Back Squat | "Progression: Paused Front Squat" but Paused Front Squat doesn't exist in DB. | Create Paused Front Squat entry or change progression to "Paused Back Squat" (already exists). Currently directs to a non-existent exercise. |
| Dumbbell Floor Press → Dumbbell Bench Press | DB Floor Press is a *regression* for shoulder safety, not a progression toward DB Bench Press. | Reverse: DB Bench Press → DB Floor Press (as regression for shoulder issues). |
| Cossack Squat → Lateral Lunge | Cossack Squat is an advanced mobility + strength movement. Lateral Lunge is easier. Progression is backwards. | Progression: Weighted Cossack or none. Regression: Lateral Lunge or Deep Squat Hold. |
| Arnold Press → Standing Dumbbell Press | Arnold Press requires more shoulder rotation and control than Standing DB Press. Progression is backwards. | Regression: Standing Dumbbell Press → Arnold Press (progression: heavier Arnold or DB Overhead Press). |
| Nordic Hamstring Curl regression → Single-Leg RDL | These are different movement patterns. Regression should be Band-Resisted Nordic or Eccentric Nordic (lowering phase only). | Regression: Band-Resisted Nordic or Eccentric Lowering Nordic. |
| Prone Y Raise → Prone T Raise | Y raises require more scapular control than T raises. T raises come first. Progression is backwards. | Fix: Prone T Raise → Prone Y Raise. |
| Wrist Flexion/Extension → Farmer's Walk | Completely different exercise. Farmer's Walk is grip + core, not a progression of wrist isolation. | Progression: Weighted Wrist Flexion/Extension or Rice Bucket work. Regression: Finger Flexor Stretch. |
| Reverse Hyperextension → Single-Leg Glute Bridge | Regression chain is too big a jump. Single-Leg Glute Bridge is a bodyweight glute exercise, Reverse Hyper is a machine-based spinal erector + hamstring exercise. | Regression: Back Extension (bilateral, bodyweight). |
| Pogo Jump → Box Jump (low) | Pogo is low amplitude reactive work at the ankle. Box Jump is a maximal hip-dominant jump. Too big a gap. | Progression: Pogo → Squat Jump → CMJ → Box Jump. Fix the chain. |

### MISSING HIGH-VALUE EXERCISES

| Exercise | Family | Why It's Missing Matters |
|----------|--------|------------------------|
| **Kettlebell Swing** | Ballistic | The single most accessible ballistic exercise for non-Olympic-lifting athletes. Used in rugby, general S&C, youth training. Glaring omission. |
| **Kettlebell Clean** | Ballistic | Safer entry to catch mechanics than barbell clean. Used in KB-focused programs. |
| **Landmine Squat** | DL Knee | Shoulder-friendly squat alternative. Common in prehab and for overhead athletes. |
| **Deficit Deadlift** | DL Hip | Standard progression from Conventional Deadlift for range-of-motion increase. |
| **Block Pull** | DL Hip | Listed in Rack Pull's progression but not in the exercise table. Add it. |
| **Single-Leg Box Squat** | SL Knee | Listed in Pistol Squat's progression but not in the table. Critical ACL return exercise. |
| **Cable Fly / Pec Fly** | Horizontal Push | Common in hypertrophy / prehab programs. |
| **Lat Prayer / Shrug** | Vertical Pull | Scapular control exercise used in prehab and overhead athlete prep. |
| **Single-Arm KB Overhead Carry** | Carry | More accessible than Waiter's Walk with dumb-bell for overhead athletes. |
| **Side Plank (leg raise)** | Core | Listed in progression but not in exercise table. |
| **Weighted Plank** | Core | Listed in RKC Plank's progression but not in table. |
| **Standing Rollout** | Core | Listed in Ab Wheel progression but not in table. |
| **Toes to Bar** | Core | Listed in Hanging Leg Raise's progression but not in table. |
| **Wall Sit** | DL Knee / Prehab | Used in ACL return, patellar rehab, team warm-ups. |

| **Split Stance RDL** | SL Hip | Bridge between bilateral RDL and Single-Leg RDL. |

| **Isometric Hamstring Hold** | DL Hip / Accessory | Used in hamstring injury return protocols. |

### LOW-VALUE EXERCISES (Consider Removing or Demoting)

| Exercise | Family | Problem |
|----------|--------|---------|
| Pogo Depth Jump | Plyometric | Rare in practice. Coaches use Depth Jump or Pogos, not both combined. Niche to the point of clutter. |
| Clapping Push-Up | Plyometric | Upper-body plyometric that's more of a novelty. Depth Push-Up is what coaches actually use. |
| Decline Push-Up | Horizontal Push | Rarely programmed. Low value for field sport athletes. |
| One-Arm Push-Up | Horizontal Push | Calisthenics showcase. Rarely prescribed in S&C programs. |
| Stir the Pot | Core | Stability ball exercise that's fallen out of favor. Band-resisted or cable-resisted core work has replaced it. |
| Reverse Hyperextension | SL Hip | Nice-to-have if machine is available. Most gyms don't have one. Low priority. |

### REORGANIZATION RECOMMENDATIONS (KEEP / MODIFY / REMOVE / ADD)

| Action | Items |
|--------|-------|
| **REMOVE** | Pogo Depth Jump, Clapping Push-Up, Decline Push-Up, One-Arm Push-Up, Stir the Pot, Wrist Flexion/Extension, Neck Flexion (demote to prehab notes rather than standalone entry) |
| **MODIFY family** | Face Pull → Accessory/Prehab. Bulgarian Split Squat → SLKD only. Single-Leg RDL → SL Hip only. Pallof Press → Core only. |
| **MODIFY progressions** | Fix all 10 incorrect progressions listed above. |
| **ADD** | Kettlebell Swing, KB Clean, Landmine Squat, Deficit Deadlift, Block Pull, Single-Leg Box Squat, Side Plank (leg raise), Weighted Plank, Standing Rollout, Toes to Bar, Wall Sit, Banded Glute Med work (clam, sidelying raise), Prone I/W Raise, Split Stance RDL, Depth Drop, Isometric Hamstring Hold, Single-Leg Bridge (elevated) |
| **KEEP** | Core 155 exercises are solid. The remaining non-duplicate, non-misassigned exercises are well-chosen. |

Why is this a blueprint? |
| 6 | Power Maintenance | 9 | 9 | 9 | 8 | Best blueprint in the catalog. Clean, minimal, credible. |

### BLUEPRINTS THAT SHOULD MERGE OR BE DELETED

| Action | Recommendation |
|--------|---------------|
| **DELETE** | Upper / Lower Split (Blueprint 5). It contradicts the 58-program research which says elite field sport coaches do NOT use body-part splits. The Coaching Patterns Summary explicitly lists this as an anti-pattern. Remove it or rename it to "Hypertrophy Split" and merge with Blueprint 11. |
| **MERGE** | Hypertrophy / Mass (11) + Upper/Lower Split (5) into a single "Hypertrophy Specialization" blueprint with a note: "For aesthetic/mass goals only. Not recommended for field sport athletes in-season." |
| **KEEP** | All others. |

### MISSING BLUEPRINTS

| Missing Blueprint | Why Needed |
|-------------------|------------|
| **In-Season Rugby** | Rugby off-season exists but in-season is different — 2 sessions/week, contact maintenance, position-specific loading. Currently covered by Power Maintenance but rugby needs collision prep even in-season. |
| **Youth-to-Senior Transition (U18-U20)** | Gap between Youth Foundation (U17 max) and senior blueprints. U19 cricketers, U20 rugby players need a specific bridge. |
| **Olympic Lifting Foundation** | For athletes whose primary power development is through weightlifting. Track & field throwers, some rugby programs. Currently absorbed into Strength + Power but not explicit. |

## 3. Substitution Matrix Audit

### TECHNICAL PROGRESSION VALIDATION

| Ladder | Verdict | Issues |
|--------|---------|--------|
| Knee Dominant | **Credible** | Clean. L1-L5 progression makes sense. |
| Hip Dominant | **Mostly credible** | L4 "45° Back Extension (single leg)" as direct pair for Barbell Good Morning is a stretch — they load different tissues. Better pair: RDL for both double and single leg. |
| Push Ladder | **Problematic** | Combining horizontal + vertical into one table makes this harder to use than separate ladders. A coach looking for a pushing regression wants one plane at a time. Split into two tables. |
| Pull Ladder | **Same issue** | Same problem as Push. Split into Horizontal Pull and Vertical Pull ladders. |
| Plyometric | **Credible** | Clean progression. Well-structured. |
| Ballistic | **Credible but incomplete** | No Kettlebell Swing or KB Clean anywhere. For the 80% of athletes who don't do Olympic lifting, this ladder offers only Med Ball work. Add a "KB Track" column. |
| Rotational | **Credible** | Clean. Anti-rotation vs loaded rotation distinction is well done. |
| Core | **Credible** | Clean. Three-column format (anti-ext/anti-rot/flexion) is smart. |
| Sprint / COD | **Credible** | Good. Acceleration / Max V / COD columns are useful. |
| Carry | **Credible** | Clean. |

### SUBSTITUTIONS A COACH WOULD REJECT

| Substitution | Why Rejected |
|-------------|-------------|
| **Vertical Push → Horizontal Push (incline bench)** | Cross-family table says this is the best swap for missing vertical push. A coach would say these are different planes with different shoulder mechanics. Acceptable as an emergency-only swap but the note already acknowledges this ("weak swap"). Accept with current note. |
| **Nordic Hamstring Curl → Single-Leg RDL** | Listed as regression in the ontology but these are different exercises for different goals (Nordic is eccentric hamstring isolation; SLRDL is a hinge). Regression should be Band Nordic or Eccentric Lowering. |
| **Barbell Good Morning → RDL** | Listed as regression. These are different exercises loading different regions (GM loads erector + hamstring at top; RDL loads hamstring + glute at bottom). Acceptable as a regression but should note the shift in load distribution. |
| **Stir the Pot → Ab Wheel Rollout** | Progression says Stir the Pot → Ab Wheel. Most coaches would not use Stir the Pot as a setup for Ab Wheel work. They'd go Plank → Ab Wheel (kneeling) → Standing Rollout. Stir the Pot is an optional progression path, not the primary one. |

### INJURY SUBSTITUTION QUALITY

| Condition | Verdict |
|-----------|---------|
| Low back pain | Good recommendations. Trap Bar Deadlift, Goblet Squat, Hip Thrust are standard. |
| Patellar tendinopathy | Good. Pogo + squat jumps + isometric holds is evidence-based. |
| Hamstring strain | Good. Isometric → concentric → eccentric progression is standard. |
| ACL return | Good. Linear → deceleration → COD → reactive COD timeline is evidence-based. |
| Shoulder impingement | Good. Landmine Press, Floor Press, Neutral-Grip DB are correct. |
| Achilles tendinopathy | Good. Heavy slow resistance (calf raises) + pogo is evidence-based. |

## 4. Coverage Audit

| Athlete | Covered? | Which Blueprint | Issues |
|---------|----------|----------------|--------|
| **U16 athlete** | YES | Youth Foundation (B7) | Adequate. Add sport-specific note for multi-sport vs single-sport U16s. |
| **U19 cricketer** | PARTIAL | Youth Foundation (B7) or Full Body Strength (B1) | Gap between U17 and senior. U19 cricketers need higher volume and rotational power work than U16s but less than senior pro. A Youth-to-Senior transition blueprint is missing. |
| **Senior cricketer** | YES | Court Sport AD (B8) modified, or Strength + Power (B2), or Strength + Cond (B3) | Adequate. Cricket-specific rotational emphasis is covered in the exercise ontology. |
| **Rugby forward** | PARTIAL | Rugby Off-Season (B9) | No position-specific guidance. Forward (tight 5) needs more neck, carry, and scrum-specific work than this blueprint provides. The ontology has Neck Flexion but it's a single entry. Scrum-specific work is absent. |
| **Rugby back** | PARTIAL | Rugby Off-Season (B9) or Sprint Development (B10) | No position-specific guidance. Backs need more speed, COD, and less collision prep than forwards. Currently both use the same blueprint. |
| **Soccer midfielder** | YES | Strength + Conditioning (B3) or Power Maintenance (B6) or Mixed Modal GPP (B14) | Adequate. The soccer midfielder is well-served by multiple blueprints. COD work is highest in soccer and the ontology covers this. |
| **Tennis player** | YES | Court Sport AD (B8) | Adequate but missing tennis-specific notes on surface programming (hard court vs clay) and tournament block periodization. The Pattern Library has this data but it's not in the blueprint. |
| **Badminton player** | YES | Court Sport AD (B8) | Adequate but missing complex training note (strength + plyometric alternated in same session) which badminton research specifically supports. |

**Overall verdict:** YES, with caveats. The three file system covers all 8 athlete types at a usable level. U19 cricketer transition, rugby position-specific loading, tennis surface programming, and badminton complex training are gaps that need bridging but do not require architecture changes — they require additions to existing files.

## 5. Final Verdict

### Scores

| Dimension | Score | Rationale |
|-----------|:-----:|-----------|
| **Knowledge Quality** | **7.5/10** | Good breadth of 159 exercises across 15 families. Strong research foundation in Pattern Library and Frequency Report. However, 10+ incorrect progression/regression chains, duplicate entries, and one major family misassignment (Face Pull) reduce confidence. No Kettlebell Swing is a surprising gap. |
| **Coaching Credibility** | **7/10** | The architecture (blueprints, families, substitution) is credible. A coach would recognise the session structures. But the progression errors, combined pull/push ladders, and absence of position-specific rugby guidance would cause a coach to say "this needs cleaning up." The Upper/Lower Split blueprint existing alongside research that calls it an anti-pattern undermines credibility. |
| **MVP Readiness** | **7/10** | Releaseable as a v1 knowledge base. The core ideas are sound. But the 10+ progression errors and ontology inconsistencies would be the first things coaches report. Fix those and this jumps to 8.5. |

### "If FORGE were released tomorrow using only these files, what would coaches complain about first?"

In order of likelihood:

1. **"Face Pull is under Vertical Push."** Every coach will see this immediately. It's the most visible error.

2. **"Where's the Kettlebell Swing?"** The single most requested exercise that's missing. Coaches at every level use it.

3. **"Bulgarian Split Squat is listed twice — which one do I use?"** Duplicate entries undermine trust in the data.

4. **"The progressions are backwards on half the exercises."** Hack Squat → Back Squat, Arnold Press → Standing Press, Prone Y → Prone T, Wrist Flexion → Farmer's Walk. A coach finds one of these and questions the entire dataset.

5. **"Why is Upper/Lower Split a blueprint when elite coaches don't use it?"** The research file explicitly identifies body-part splits as an anti-pattern, yet the catalog features it as a primary blueprint. Inconsistent.

6. **"How do I program for a rugby forward versus a back?"** The Rugby Off-Season blueprint treats all rugby players identically. A forward needs scrum prep, neck work, and heavy carries. A back needs speed, COD, and less mass. Position-specific guidance is absent.

7. **"The Push Ladder combines horizontal and vertical — I need to see them separately."** Coaches think in planes. Combining them into one table creates friction.

8. **"Pogo Jump → Box Jump is too big a gap."** Missing intermediate steps (Squat Jump, CMJ) break the progression logic.

The errors are in the detail — progressions, assignments, gaps — not in the structure. Fix the P0 items and it's MVP-ready. The Pattern Library and Frequency Report are the strongest files in this set. The Exercise Ontology needs the most cleanup.
