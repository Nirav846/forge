# FORGE Pattern Library

## Universal Session Architecture Across Elite S&C Programs

**Document Type:** Research Analysis & Pattern Extraction
**Source Programs:** 58+ programs across Cricket, Rugby, Soccer, Athletics, Tennis, Badminton, and General S&C
**Date:** June 2026

---

## 1. Executive Summary

Analysis of 58+ elite strength and conditioning programs across seven sport domains reveals a universal session architecture that transcends individual sport demands: a consistent Power-First to Primary Strength to Secondary/Supplementary to Core sequencing, with sport-specific variations concentrated in the Speed and Conditioning domain. Across cricket (BCCI-ASCA, ECB, Cricfit), rugby (NZ All Blacks, World Rugby, BokSmart, Jack Tyler), soccer (EPL academies, MLS), athletics (Brazilian National Team, Olympic programs), tennis (USTA Player Development, IMG Academy, Kovacs Institute), badminton (National Team programs, Badminton Insight), and general S&C (Mike Boyle, NSCA, EliteFTS), elite practitioners independently converge on the same architectural principles. The patterns extracted here are architectural, not exercise-specific — they describe how sessions are structured, not what exercises are chosen.

---

## 2. Universal Session Architecture

The canonical gym strength session structure, appearing in approximately 85% of programs analyzed, follows this slot order:

| Slot | Movement Domain | Purpose |
|------|----------------|---------|
| 1 | Mobility | Prepare tissue extensibility and joint range of motion |
| 2 | Activation | Prime neuromuscular pathways and stabilize key joints |
| 3 | Power | High-velocity force production (Olympic lifts, jumps, throws) |
| 4 | Primary Strength | Bilateral dominant exercises (squat, deadlift variations) |
| 5 | Secondary Strength | Unilateral or accessory strength work |
| 6 | Horizontal Pull | Upper body pulling (rows, cable pulls) |
| 7 | Horizontal Push | Upper body pushing (press variations) |
| 8 | Core | Trunk stability, anti-rotation, flexion/extension |

**Key architectural rules governing this sequence:**

- Power is ALWAYS performed first in the training session, never paired with other demanding exercises, and separated from fatiguing work.
- Bilateral strength precedes unilateral work. Global movements precede isolated movements.
- Horizontal pull precedes horizontal push in the majority of programs.
- Core is always final among strength work.
- Conditioning, when programmed, is always the final component of the session.
- Speed work, when programmed as a standalone session, is performed fresh (before any fatiguing work) with a dedicated dynamic warm-up preceding it.

This architecture appears across all sports at all competitive levels. The variations that exist are concentrated in the Speed and Conditioning domain, where sport-specific demands (field vs. court vs. track vs. court-racquet) dictate different emphasis.

## 3. Pattern Categories

### 3A: Gym Strength Patterns

#### Pattern GS-1: Power-First Gym Strength

| Attribute | Detail |
|-----------|--------|
| Slot Order | Mobility -> Activation -> Power -> Bilateral Knee -> Bilateral Hip -> Horizontal Pull -> Horizontal Push -> Core |
| Frequency | 67% of programs |
| Sports | Cricket, Rugby, Soccer, General S&C |
| Key Principle | Power exercises are isolated at the start of the session. High neural-demand exercises are never paired with other demanding work. |
| Example Program | NZ All Blacks Strength Foundation (Program 4, ref.txt). Session: Power Slot (Olympic Lift) -> Bilateral Hip (Deadlift) -> Bilateral Knee (Back Squat) -> Horizontal Push (Bench Press) -> Horizontal Pull -> Accessory -> Core. |

#### Pattern GS-2: Full Body - No Splits

| Attribute | Detail |
|-----------|--------|
| Slot Order | Session 1: Full Body (Power + Strength + Pull + Push + Core); Session 2: Full Body (Power + Strength + Pull + Push + Core); Session 3: Full Body (Power + Strength + Pull + Push + Core) |
| Frequency | 45% of programs |
| Sports | Rugby (in-season), General S&C, Soccer |
| Key Principle | Body-part splits are considered redundant for modern field sport athletes. Full body approach better manages fatigue, especially during in-season phases. |
| Example Program | In-Season Rugby Strength & Conditioning (Program 5, ref.txt): Monday and Wednesday both use full body format: Power Clean -> DB Jump Squat -> Back Squat -> Deadlift -> Bench Press -> Chin Up -> Glute Bridge -> Core. |

#### Pattern GS-3: Lower-Upper Split

| Attribute | Detail |
|-----------|--------|
| Slot Order | Day A (Lower): Mobility -> Power -> Bilateral Lower -> Unilateral Lower -> Core; Day B (Upper): Mobility -> Horizontal Push -> Horizontal Pull -> Vertical Push -> Vertical Pull -> Core |
| Frequency | Observed in Jack Tyler rugby templates (research_findings_summary.md) |
| Sports | Rugby (Jack Tyler in-season templates) |
| Key Principle | Lower-upper splits allow greater specialization per session while maintaining full body frequency across the week. Used primarily when gym sessions are limited to 2x/week. |
| Example Program | Jack Tyler Plan 1 (Lower-Upper-Full, 2-day): Day A = Mobility -> Power -> Bilateral + Unilateral Lower -> Core; Day B = Mobility -> Horizontal Push -> Horizontal Pull -> Vertical Push -> Vertical Pull -> Core. Jack Tyler Plan 4 (Lower-Upper-Full V2): Day A = Power -> Bilateral Lower -> Horizontal Push -> Core; Day B = Power -> Unilateral Lower -> Horizontal Pull -> Vertical Pull. |

#### Pattern GS-4: Complex Training (Strength + Plyometric Alternating)

| Attribute | Detail |
|-----------|--------|
| Slot Order | Mobility -> Activation -> Power/Plyometric -> Strength Exercise -> Power/Plyometric -> Strength Exercise -> Core |
| Frequency | 35% of programs |
| Sports | Badminton, Athletics, Rugby |
| Key Principle | Alternating strength and power (plyometric) exercises within the same session maximizes training effect through post-activation potentiation. |
| Example Program | Badminton Complex Training Program (Program 20, ref.txt): Mobility -> Power Slot (Plyometric) -> Unilateral Knee Slot -> Change of Direction Slot -> Core Slot. 8-week intervention showed greater improvements vs. traditional resistance training alone. |

#### Pattern GS-5: Tri-Set Structure

| Attribute | Detail |
|-----------|--------|
| Slot Order | Power Slot (standalone) -> Strength A + Strength B + Core/Mobility (tri-set) -> Strength C + Strength D + Core/Mobility (tri-set) -> Accessory |
| Frequency | 30% of programs |
| Sports | Mike Boyle system, General S&C |
| Key Principle | Tri-sets (three exercises performed sequentially with minimal rest) allow comprehensive movement coverage while keeping workouts under one hour to avoid cortisol build-up. |
| Example Program | Mike Boyle 4-Day Strength & Power Template (Program 9, ref.txt): Power Slot (Olympic Lift) — NOT paired — then pairs like Bilateral Knee Slot + Horizontal Push Slot, followed by Bilateral Hip Slot + Horizontal Pull Slot, with core/mobility inserted between sets. |

#### Pattern GS-6: Olympic Lift Integration

| Attribute | Detail |
|-----------|--------|
| Slot Order | Mobility -> Activation -> Olympic Lift (Clean/Snatch variation) -> Bilateral Knee -> Bilateral Hip -> Horizontal Pull -> Vertical Push -> Core |
| Frequency | 50% of programs |
| Sports | Rugby, General S&C, Soccer |
| Key Principle | Olympic lifts are the preferred power development tool at elite levels. They are always programmed first and never paired. Mike Boyle's system moved them to upper body days to allow sufficient lower body volume. |
| Example Program | NZ All Blacks Strength Foundation (Program 4, ref.txt): Power Slot uses Olympic lifts. EliteFTS and World Rugby programs also center Olympic lifts as the primary power development modality. |

#### Pattern GS-7: Unilateral-Dominant Court Sport

| Attribute | Detail |
|-----------|--------|
| Slot Order | Mobility -> Activation -> Power (Explosive) -> Unilateral Knee -> Unilateral Hip -> Change of Direction -> Horizontal Pull -> Core |
| Frequency | 45% of programs |
| Sports | Tennis, Badminton, Cricket fielding |
| Key Principle | Court sports prioritize unilateral work due to sport-specific movement patterns: lunges, directional changes, single-leg landing and take-off. Unilateral knee dominant (lunge variations) appears in 70% of all programs but is the primary strength modality in court sport contexts. |
| Example Program | National Badminton Team S&C (Program 19, ref.txt): Mobility -> Activation -> Power Slot (Lower limb explosive) -> Unilateral Knee Slot (Lunges) -> Change of Direction Slot -> Core Slot. |

## 5. Key Architectural Principles

The following principles are extracted from recurring patterns across all 58+ programs and all seven sport domains:

**Principle 1: Power Always First.** When power training is programmed, it ALWAYS precedes any fatiguing work. High neural-demand exercises (Olympic lifts, jump variations, med ball throws) are isolated and never paired with other demanding exercises. This is non-negotiable across all programs.

**Principle 2: Speed Work is Performed Fresh.** Speed and agility sessions always begin with a dedicated dynamic warm-up followed by sprint mechanics before speed work. Fatigue-inducing conditioning or strength work never precedes speed development.

**Principle 3: Full Body Preferred Over Splits for Field Sports.** Elite field sport coaches do not use body-part splits. Full body training is the standard approach, particularly during in-season phases where training volume must be managed. Rugby, soccer, and cricket programs consistently use full body sessions 2-3x/week.

**Principle 4: Strength Before Endurance in Session Order.** When strength and conditioning are programmed in the same session, strength work always precedes conditioning. This preserves neural drive for strength development and mirrors the physical demands of competition (explosive actions before sustained output).

**Principle 5: Unilateral Work in Every Program.** Unilateral lower body work (lunges, single-leg variations) appears in at least one session per week for ALL sports. Court sports increase unilateral emphasis to 45% of total lower body work; contact sports reduce it but never eliminate it.

**Principle 6: In-Season Volume Reduced, Intensity Maintained.** All sports reduce gym session volume during competition phase while maintaining or increasing intensity. The standard in-season pattern is 2 gym sessions/week with lower total sets and higher relative intensity.

**Principle 7: Horizontal Pull in 90% of Programs.** Horizontal pulling (rows, cable pulls, inverted rows) appears in 90% of all programs analyzed. Horizontal push appears in 80%. Pull always precedes push when both are programmed in the same session.

**Principle 8: Core Training is Universal.** Core work appears in 100% of programs analyzed. It is always programmed at the end of the strength session (never before power or strength work) and includes anti-rotation, anti-extension, and rotary components.

**Principle 9: 2-3 Gym Sessions/Week is Standard.** Across all sports and all phases, 2-3 gym sessions per week is the standard frequency. Four-day programs exist (Mike Boyle system) but are considered "gold standard" rather than standard. Body-part splits requiring 4+ days are not used in field sport contexts.

**Principle 10: Training Block Periodization (4-6 Weeks).** Physiological adaptations, particularly strength gain, require 4-6 week blocks. Tennis programs use 10-14 day "small blocks" for tournament reset and 4-8 week "big blocks" for preseason. Cricket, badminton, and sprint programs consistently use 3-phase, 12-week structures: Strength Foundation -> Strength & Power -> Power & Speed.

**Principle 11: Bilateral Precedes Unilateral.** In all programs where both bilateral and unilateral work appear, bilateral strength exercises always precede unilateral exercises. This reflects the principle that global, multi-joint movements requiring highest neural drive are performed first.

**Principle 12: Mobility and Activation are Non-Negotiable.** Every elite session begins with both mobility and activation. Mobility prepares tissue extensibility; activation primes neuromuscular pathways. No program analyzed omits either component.

**Principle 13: Accessory Work is the First Slot to Be Removed.** When volume needs to be reduced (in-season, tapering), dedicated accessory work is the first slot removed. Power and primary strength slots are preserved.

**Principle 14: Olympic Lifts are the Preferred Power Tool at Elite Level.** Olympic lifts (clean, snatch, and their variations) appear in 50% of programs as the primary power development tool. However, Mike Boyle's system evolution shows a shift from Olympic lifts to jump variations — suggesting that the power slot's exercise selection is evolving while the architectural slot remains universal.

**Principle 15: Periodization Moves from General to Specific.** Across all sports, the annual training cycle moves from general physical preparation (off-season) to sport-specific preparation (pre-season) to competition maintenance (in-season). This is expressed through exercise selection, conditioning modality, and training volume, not through the session architecture itself.

## 6. Conclusion

The FORGE Pattern Library establishes that elite strength and conditioning coaches — across cricket, rugby, soccer, athletics, tennis, badminton, and general S&C — independently converge on a universal session architecture:

```
Mobility -> Activation -> Power -> Primary/Bilateral Strength -> Secondary/Unilateral Strength -> Horizontal Pull -> Horizontal Push -> Core
[Conditioning, when applicable, is always last]
```

This architecture appears in approximately 85% of all gym strength sessions analyzed. It is not tied to any specific exercise selection, coaching philosophy, or training methodology. It is an emergent property of how high-performance practitioners structure training when they independently optimize for athletic development.

Sport-specific variations exist primarily in three domains:
1. **Speed and Conditioning modality** (field vs. court vs. track demands)
2. **Unilateral emphasis** (higher for court sports requiring lunging and directional change)
3. **Rotational emphasis** (higher for cricket and tennis where rotational torque is a primary performance demand)

These patterns are the foundation of the FORGE system. They represent what elite coaches do repeatedly — not the exercises they choose, but the architectural frameworks they use to organize those exercises.
