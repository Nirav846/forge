## How Patterns Enter This Database

1. Source identified → cataloged in FORGE_SOURCE_CATALOG.md
2. Extraction performed → FORGE_PATTERN_EXTRACTION_TEMPLATE.md format
3. Frequency tallied → multiple sources checked for same pattern
4. Contradictions flagged → cross-referenced with existing entries
5. Consensus classified → FORGE_CONSENSUS_ENGINE.md determines category
6. Confidence set → Universal / Common / Situational / Rare
