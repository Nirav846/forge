# FORGE Exercise Ontology V1

> Foundational exercise library for the FORGE coaching platform.
> 155 exercises across 15 families. Minimum viable set — add nodes as sport-specific demand emerges.

---

## How To Read

| Field | Meaning |
|-------|---------|
| **Family** | The biomechanical / movement category |
| **Difficulty** | B = Beginner, I = Intermediate, A = Advanced |
| **Progression** | What comes *after* this exercise in the same family |
| **Regression** | What comes *before* this exercise in the same family |
